WEBSTART
========
前端项目基础模板

有关使用说明，请查看 [WIKI](https://github.com/changyou/webstart/wiki)

## 使用步骤

1. 下载代码
  
  $> git clone https://github.com/changyou/webstart.git

2. 安装FED:

  $> npm install -g fed

3. 启动：

  $> cd webstart  
  $> fedmon run ./seaconfig.js

4. 访问：

  http://localhost:3000
